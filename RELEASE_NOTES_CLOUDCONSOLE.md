# Checkly Lists Release Notes

## Version 1.0.0 (Initial Release)

**Fresh Google Play Release - Checkly Lists**

- **New Branding**: Rebranded application to "Checkly Lists" with package name `com.checkly.lists` for clear market positioning.
- **16 KB Memory Page Size Support**: Full compatibility with devices using 16 KB memory page sizes through proper memory tagging and ARM64 ABI filtering.
- **Offline-First Architecture**: Complete offline-first functionality with Hive local storage for reusable lists and daily routines.
- **Minimal Permissions**: Clean permission model - only requests POST_NOTIFICATIONS for reminders, no unnecessary permissions or data collection.
- **Privacy Focused**: No ads, no analytics, no tracking, no data sharing. All data remains on user's device.
- **Google Play Compliant**: Fully compliant with Google Play Store policies including content rating, privacy requirements, and technical standards.
- **Production Ready**: Stable release with Material 3 design, dark mode support, and full notification functionality.
